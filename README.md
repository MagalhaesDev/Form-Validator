# Validating form

System for validating a form

## Technology 

Here are the technologies used in this project.

* Typescript 4.6.3
* Eslint 8.14.0
* Prettier 2.6.2
* Webpack 5.72.0

## Services Used

* Github.

## Getting started
 
1 - Run the command npm install in the terminal to download the dependencies.

2 - npm run build:frontend, to generate a server.
